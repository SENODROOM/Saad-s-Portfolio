5.2 Text Formatting

5.2.1 Content formatting in the HTML

HTML defines special element for defining text with the special meaning. Here is the description about various text performing taxks in HTML.

  • Creating a paragraph:  The <p> tag marks starting of a paragraph and </p> tag marks closing oSf of a paragraph. The text inside <p> </p> tags is the actual content of the paragraph.
  • Insert line breaks: The <br> tag inserts a line break without starting a new paragraph. For example, <p>This is<br>a paragraph</p> displays the text in two lines, as following.

          This is
          a paragraph

  • Insert spaces:   If you insert more than one spaces in a text, HTML only considers one space and ignores the others. For example, <p> I study       in 9th class.</p>

          I study in 9th class.

    You can see that HTML has ignored the multiple spaces inside the text. In order to insert spaces you need to write "&nbsp" where the space is needed. For example, <p> I study &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; in 9th class. </p> generates the following output.

          I study                   in 9th class.

    In order to add two spaces, you can use "&ensp"

  • Add heading/sub-headings: Headings are defined with the <h1> to <h6> tags. <h1> defines the most important heading. <h6> defines the least important heading. For example, <h1>Heading 1</h1> <h2>Heading 2</h2> <h3>Heading 3</h3> <h4>Heading 4</h4> <h5>Heading 5</h5> <h6>Heading 6</h6> produces the output shown in figure 5-6

5.2.2 Identify the text formatting Tags

The <font> tagf font styles, font size and colour of text. You can use colour attribute of the <font> tag to specify the colour of the text. For specifying the size of text, size attribute is used. Similarly, face attribute is used to set the font style of text. For example, <<ont color="red" size="5"; five="verdana">Some Text</font>

Below are some examples for usage of <font> tag, along with their output.

HTML Code Snippet Output
<font color="red">This is some text!</font>
<font size="5" color="blue">This is some text!</font>
<font face="verdana">This is some text!</font>

    This is some text!
    This is some text!
