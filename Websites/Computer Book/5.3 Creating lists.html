5.3 Creating Lists

Somethimes we need to proide information in the form of lists e.g. list of subjects, list of teachers and list of students. In the following, we will what type of lists are available in HTML and how to create them

5.3.1 Types of lists

there are following three types of lists in HTML

  • Unordered List

    In an unordered list the order of the list item is not important. In other words, shuffling of items in an unordered list has no effect. For example, list of cities in pakistan. An unordered list is created inside the <ul></ul> tags. Each list item is added with <li> tag, as shown below.

    HTML Code Snippet Output
     <ul>
             <li>Item</li>
             <li>Item</li>
             <li>Item</li>
             <li>Item</li>
      </ul>
    • Item        
    • Item
    • Item
    • Item

  • Ordered list

    An ordered list keeps the item with an order number. If you change, the order, the meaning of the whole list may also change. For example, if your teacher makes the list of students with respect to thier marks then definitely order will matter

    An ordered list starts with <ol> and ends with </ol> tag. Each list item starts with <li> tag, as shown below.

    HTML Code Snippet Output
     <ol>
             <li>First Item</li>
             <li>Second Item</li>
             <li>Third Item</li>
             <li>Fourth Item</li>
      </ol>
    1. Item        
    2. Item
    3. Item
    4. Item

  • Definition list

    There is another type of list, called "Definition" or "Description list". It is used when you need to show terms and their description. For Example, If you want to write the names of subjects you are studying in 9th class along with their introductions then this type of list is helpful. The <dl> describes the description lisdt, the <dt> tag specifies the term, and the <dd> tag describes that term as shown in the following.

    HTML Code Snippet Output
     <dl>
             <dt>Coffee</dt>
             <dd>-black hot drink<dd>
             <dt>Milk</dt>
             <dd>-white cold drink<dd>
      </dl>
    Coffee
    -black hot drink
    Milk
    -white cold drink

  • Nested List

    In a list, a list item can contain another list. Such a list is called a nested list. It is useful for a list where you hse multiple options for a single item in a list e.g. for writing table of contents that contain sub sections. For example, see the following code of snippet and its output.

    HTML Code Snippet Output
     <ul>
             <li>Coffee</li>
             <li>Tea
                  <ul>
                              <li>Black coffee</li>
                              <li>Green tea</li>
                  <ul>
             </li>
             <li>Milk</li>
      </ul>
    • Coffee
    • Tea
      • Black coffee
      • Green tea
    • Milk
